## Analyse Plugin

These are the official Minecraft adapters for [Analyse](https://analyse.net) - the stress-free way to track your Minecraft server analytics. With no open ports or extra server memory needed!

## Installation

1. Download the latest version from our [releases page](https://github.com/track/plugin/releases).
2. [Sign in](https://app.analyse.net/register) to the dashboard.
3. Click to create a new server and follow the setup wizard.


## Compiling Analyse
1. Clone this repo (`git clone git@github.com:track/plugin.git`).
2. Make any needed changes.
3. Compile using `./gradlew`.
